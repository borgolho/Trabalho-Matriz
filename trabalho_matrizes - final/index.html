Trabalho de Matrizes WEB II e Matemática II

Matrizes são elementos da matemática que tem como objetivo a organização de dados em linhas e colunas. Como um elemento matemático é possível realizar operações matemáticas entre elas, como adição, subtração, multiplicação, entre outras.

Como qualquer outro elemento as matrizes possuem regras rigorosas durante suas operações. O processo de adição e subtração, por exemplo só podem ser realizadas quando ambas matrizes possuem os mesmos números de linhas e colunas.
